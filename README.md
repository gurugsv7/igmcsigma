### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/gurugsv7/igmcsigma.git
   cd igmcsigma
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

   ```bash
   npm run build
   ```

5. **Preview production build**
   ```bash
   npm run preview
   ```
